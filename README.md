# Virat Kohli Performance Analysis Dashboard

## Project Overview
This project analyzes the ODI and T20 batting performance of Virat Kohli using interactive data visualization in Power BI.

The dashboard helps identify:\
Performance trends across years\
Best grounds and opponents\
Monthly and quarterly performance insights\
Total runs and key batting statistics

The goal of this project is to transform raw cricket data into actionable insights using data analytics techniques.

 ## Business Questions Solved
1️⃣ Top 5 Ground Performance (2021)\
Analyze Virat Kohli’s top 5 grounds based on total runs scored in 2021.


2️⃣ Monthly & Quarterly Performance (2018)\
Analyze batting performance during 2018 to identify:\
Best months in terms of runs scored\
Best quarters of the year\
Time periods where Virat delivered peak performances

3️⃣ Best Opposition Performance (2021)\
Identify which opposition teams Virat performed best against in 2021.\
Metrics used:\
Maximum runs scored\
Frequency of high scoring performances

4️⃣ Match Participation Analysis\
Determine how many matches Virat played for India from :\
2008-2022\
This helps analyze workload and participation trends across years.

 ## Key Dashboard Metrics
The dashboard provides the following KPIs:\
Total Runs: 23,709\
Maximum Runs in a Match: 254\
Average Run Rate: 45.95\
Opponent Matches: 516

## Dashboard Features
Interactive components include:\
Year filter (2008–2022)\
Ground-wise performance bar charts\
Opponent performance table\
Monthly and quarterly analysis

## Conclusion
This dashboard demonstrates how data analytics can be used in sports performance evaluation.\
By analyzing historical performance data, we can easily identify:\
Strong venues
Favorable opponents
Performance peaks over time
